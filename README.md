# WardenCloud G3n Bot — Guide de démarrage

## 1. Fichiers vides à créer (stock)
Ces fichiers doivent exister (même vides) :
- stock/Minecraft.txt
- stock/Steam.txt
- stock/Crunchyroll.txt
- stock/Mc_Bedrock.txt
- stock/Xbox.txt
- stock/Cape.txt
- bosststock/Donut.txt
- bosststock/Unbanned.txt
- paidstock/Mcfa.txt

## 2. Remplir config.json
Remplace chaque valeur "ID_..." par le vrai ID Discord.

## 3. Déploiement Railway
1. Mets le dossier sur GitHub (privé !)
2. railway.app → New Project → GitHub
3. Variables : DISCORD_TOKEN = ton token
4. Le bot démarre automatiquement

## 4. Commandes
Voir $help dans Discord.
